El objetivo modificar es crear una función auxiliar seasonder_CSSY2CSHeader que será llamada por seasonder_readSeaSondeRCSSYFile para transformar el header del fichero CSSY en un header valido para un objeto SeaSondeRCS. El componente cs4h del fichero CSSY tiene el formato adecuado. Lo que vamos a hacer es extraer ese componente para el header_cs. Luego eliminaremos el componente cs4h del header original y finalmente asignaremos el header original sin el elemento cs4h a un elemento header_cssy del header_cs.
